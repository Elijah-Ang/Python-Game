# Parameters & Arguments

**Concepts:**
- Inputs to functions
- Multiple parameters

**Why it matters:** Parameters let you turn one idea into many useful tools.

## Steps
1. Define a function with parameters.
2. Call it with different values.
3. Print results.

## Try it
Write `def add(a, b):` that prints `a + b`.
Call it with (3, 4) and (10, -2).
