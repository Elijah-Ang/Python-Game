# Searching: Linear vs Binary

**Concepts:**
- Linear search scans everything
- Binary search needs sorted data

**Why it matters:** Search is everywhere (finding a record, matching an id, locating an item).

## Steps
1. Implement linear search.
2. Return index or -1.
3. Test it.

## Try it
Write `linear_search(arr, target)` returning index or -1.
Test with `[4,7,1]` target 7 (should return 1).
