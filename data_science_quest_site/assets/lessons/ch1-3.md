# Numbers & Math

**Concepts:**
- Integers vs floats
- Operators: `+ - * / // % **`
- Type conversion: `int()`, `float()`

**Why it matters:** All calculations in analytics, ML, finance, and science depend on numeric thinking.

## Steps
1. Create two numbers (one int, one float).
2. Compute a few operations.
3. Convert types deliberately.

## Try it
Make `a = 7` and `b = 2`.
Print:
- `a / b`
- `a // b`
- `a % b`
Then convert `a` to float and print `float(a) / b`.
