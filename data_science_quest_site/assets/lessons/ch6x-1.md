# Modules

**Concepts:**
- `import math, random, datetime`
- Namespaces and reusable code

**Why it matters:** Modules are how you scale beyond one file and use the Python ecosystem.

## Steps
1. Import a module.
2. Use a function from it.
3. Print the result.

## Try it
Import `math`.
Print `math.sqrt(81)` and `math.pi`.
