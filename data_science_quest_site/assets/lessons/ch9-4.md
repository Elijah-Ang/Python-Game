# Recursion (Starter)

**Concepts:**
- A function calls itself
- Base case stops the recursion

**Why it matters:** Recursion appears in algorithms, trees, and certain model explanations.

## Steps
1. Write factorial recursively.
2. Test factorial(5).
3. Print the result.

## Try it
Write `fact(n)`:
- if n == 1: return 1
- else return n * fact(n-1)
Print `fact(5)`.
