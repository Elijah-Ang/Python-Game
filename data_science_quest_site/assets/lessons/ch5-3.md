# Dictionaries

**Concepts:**
- Key → value mapping
- Lookups and updates

**Why it matters:** Most real data is keyed: user_id → profile, product → price, feature → value.

## Steps
1. Create a dict.
2. Read a key.
3. Update/add a key.

## Try it
Make `prices = {"tea": 4.9, "coffee": 5.5}`.
Add `"water": 1.2`.
Print the price of tea and the number of items.
