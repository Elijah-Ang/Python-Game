# range()

**Concepts:**
- `range(start, stop, step)`
- Generate sequences

**Why it matters:** Many data tasks are index-based or require repeated actions N times.

## Steps
1. Use `range(5)`.
2. Use `range(2, 10, 2)`.
3. Use the loop variable in a calculation.

## Try it
Use `range(1, 6)` to print the squares:
1, 4, 9, 16, 25 (one per line).
