# JSON

**Concepts:**
- Structured data format
- Data interchange

**Why it matters:** JSON is everywhere: APIs, configs, logs, datasets.

## Steps
1. Create a Python dict.
2. Convert to JSON string.
3. Convert JSON back to dict.

## Try it
Create `user = {"name":"Elijah","role":"student"}`.
Use `import json`.
Print `json.dumps(user)`, then load it back with `json.loads(...)` and print the dict.
