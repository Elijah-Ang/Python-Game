# Strings

**Concepts:**
- String literals: `"text"` and `'text'`
- Concatenation: `"Hello " + name`
- F-strings: `f"Welcome {name}"`
- String methods: `.upper()`, `.lower()`, `.strip()`

**Why it matters:** Strings are everywhere: UI text, file content, labels, logs, and data cleaning.

## Steps
1. Create a string variable.
2. Use an f-string to combine text + a variable.
3. Use `.upper()` and print the result.

## Try it
Create a variable `city = "singapore"`.
Print:
- `city` in uppercase
- a sentence like `I live in Singapore` using an f-string.
