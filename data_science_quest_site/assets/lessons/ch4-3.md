# Return Values

**Concepts:**
- Return vs print
- Using returned data later

**Why it matters:** Data science code *returns* values so you can pipe results into later steps.

## Steps
1. Define a function that returns a value.
2. Store the result in a variable.
3. Print the stored result.

## Try it
Write `def area_circle(r):` that returns `3.14159 * r * r`.
Compute `area_circle(2)` and print it formatted to 2 decimals.
