# Lists

**Concepts:**
- Ordered, mutable collection
- Indexing and appending

**Why it matters:** Lists are the basic container for rows of data, features, and sequences.

## Steps
1. Create a list.
2. Append an item.
3. Access by index.

## Try it
Create `fruits = ["apple", "banana"]`.
Append `"mango"`.
Print the first and last fruit.
