# Sets

**Concepts:**
- Unique unordered elements
- De-duplication

**Why it matters:** Sets are a fast way to remove duplicates and check membership.

## Steps
1. Create a list with duplicates.
2. Convert to a set.
3. Print the unique items.

## Try it
Given `nums = [1, 1, 2, 3, 3, 3]`,
convert to a set and print it.
