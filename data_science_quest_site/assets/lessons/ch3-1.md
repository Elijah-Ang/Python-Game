# If / Elif / Else

**Concepts:**
- Conditional execution
- Branching logic

**Why it matters:** This is how software chooses actions based on data.

## Steps
1. Create a numeric variable.
2. Branch into 3 cases using if/elif/else.
3. Print which path ran.

## Try it
Set `score = 78`.
Print:
- `A` if score >= 85
- `B` if score >= 70
- `C` otherwise
